Як встановити Codex на Mac

Простий спосіб встановити Codex, увійти в акаунт і перевірити, що все працює через Terminal.

Codex на Mac зручно використовувати як застосунок і як CLI у Terminal. Якщо все встановлено правильно, ви зможете запускати Codex прямо з командного рядка, відкривати застосунок і працювати з кодом локально.

Крок 1: Встановіть Codex на Mac

Завантажте і встановіть Codex для macOS з офіційного джерела OpenAI або через інсталятор застосунку. Після встановлення у вас з'явиться застосунок Codex у папці Applications.

Якщо Codex уже встановлений, у системі зазвичай доступна команда codex у Terminal.

Крок 2: Відкрийте Terminal і перевірте CLI

Після встановлення відкрийте Terminal і виконайте:

codex --version

Якщо все добре, Terminal покаже встановлену версію Codex CLI. Наприклад, локально команда повертає щось на кшталт codex-cli ....

Для перегляду доступних команд можна виконати:

codex --help

Крок 3: Увійдіть у свій акаунт

Щоб почати працювати, зазвичай потрібно авторизуватися. Для цього використовуйте:

codex login

Після цього Codex запропонує пройти авторизацію через браузер або інший зручний спосіб входу.

Крок 4: Запустіть Codex

Після входу можна просто відкрити інтерактивний режим у Terminal:

codex

Або відразу передати короткий запит:

codex "поясни структуру цього проєкту"

Що робити, якщо Terminal не бачить команду codex

Іноді після встановлення macOS не підхоплює шлях до CLI одразу. У такому випадку допомагає:

  • повністю закрити і знову відкрити Terminal;
  • перезапустити Mac, якщо команда все одно не знаходиться;
  • відкрити сам застосунок Codex і перевірити, чи він завершив налаштування.
Порада: якщо хочете впевнитися, що macOS бачить Codex, виконайте which codex. Це покаже шлях до CLI у системі.

Коротка перевірка після встановлення

Після успішної установки достатньо пройти три прості перевірки:

  1. codex --version повертає версію.
  2. codex --help показує список команд.
  3. codex login дозволяє увійти в акаунт.

Після цього Codex на Mac готовий до роботи. Якщо захочеш, я можу ще підготувати окрему статтю про те, як користуватися Codex у Terminal на Mac після встановлення.
